# multiplicar# Calculadora de Precio - Laptop Gamer AMD

Tarjeta de producto que calcula el **total a pagar** según la cantidad seleccionada.

## ¿Qué hace?

- Muestra el precio base del producto
- Permite aumentar o disminuir la cantidad con botones `+` y `-`
- Calcula y muestra automáticamente el total (`cantidad × precio base`)

## Archivos

| Archivo | Descripción |
|---|---|
| `index.html` | Estructura de la tarjeta y el total |
| `assets/ccs/style.css` | Estilos visuales |
| `assets/script.js` | Lógica del precio y cálculo del total |

## Uso

Abrir `index.html` en el navegador. No requiere instalación.
